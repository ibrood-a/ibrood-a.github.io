My Exploration of GitHub Profile

Published on: October 10, 2023

As a computer science major working towards my degree, I am always eager to learn and explore the work of fellow programmers. I recently came across Jacob Kennedy, who has a fascinating GitHub profile. Let's delve into some of the intriguing repositories I found on this GitHub profile:

cosc2436-lab10 (C++)

I was immediately drawn to a C++ project. This repository likely contains code related to my coursework or personal projects in C++.

T-Mobile-price-plan-calculator (HTML)

What caught my eye even more was a project I created to calculate T-Mobile plan pricing. This tool is designed to help me and my co-workers calculate plan prices accurately, as the tools provided by our company are often unreliable.

Ackermann-ARM-Assembly (Assembly)

Another fascinating repository, this one is related to ARM Assembly programming. It showcases my skills in low-level programming and adds to the diversity of my GitHub work.

Guessing-Game-ARM-Assembly (Assembly)

My curiosity led me to yet another ARM Assembly project, which appears to be a guessing game implemented in Assembly. It clearly demonstrates my expertise in creating programs in Assembly language.

Binary-Search-Tree (C++)

I have a repository dedicated to a Binary Search Tree implemented in C++. This is an excellent example of my proficiency in data structures and algorithms, which is an area I find particularly exciting.

Radix-Sort (C++)

Another repository focusing on Radix Sort implemented in C++. Radix Sort is a fundamental sorting algorithm, and this project highlights my skills in algorithm design and implementation.

If you're passionate about computer science and programming like me, I encourage you to explore my GitHub profile further to dive into these repositories and appreciate the skills and knowledge I bring to the table.
